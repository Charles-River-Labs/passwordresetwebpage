Réinitialisation de mot de passe – FAQ

Q: Je n'ai pas enregistré mais j'ai besoin de réinitialiser mon mot de passe, comment puis-je le faire?
R: Malheureusement, vous ne pourrez pas utiliser ce service; veuillez contacter le service informatique.
Q: Quel est mon nom d'utilisateur?
A: Votre nom d'utilisateur est votre identifiant CR suivi de @charlesriverlabs.com, par exemple cr121212@charlesriverlabs.com
Q: Dois-je utiliser l’interface de réinitialisation de mot de passe?
R: Non, l’utilisation n'est pas obligatoire; C'est un moyen pour vous d'avoir la possibilité de réinitialiser votre mot de passe réseau sans avoir à appeler le service informatique.
Q: Puis-je changer mon mot de passe depuis n'importe quel ordinateur?
R: Oui, vous pouvez changer votre mot de passe depuis n'importe quel ordinateur ou navigateur Web mobile.
Q: Comment fonctionne l’outil de réinitialisation de mot de passe?
R:Tout d'abord, vous devrez vous inscrire sur l’interface en choisissant au moins une méthode de vérification. Plus tard, lorsque vous tenterez de réinitialiser votre mot de passe, vous serez invité à vérifier votre identité. Une fois que l'identité a été confirmée, l’outil réinitialisera votre mot de passe et débloquera votre compte d'utilisateur.
Q: Puis-je utiliser l’interface pour changer tous mes mots de passe?
A: Le serveur de réinitialisation de mot de passe ne changera que les mots de passe dans le domaine CR, qui est dans certains cas affiché sous le nom de domaine CR.Local. C'est le domaine utilisé pour se connecter à votre ordinateur. Bien que de nombreux systèmes utilisent vos informations d'identification windows pour permettre l'accès à l'application, d'autres, comme SAP ne le font pas (même si nous avons fourni un autre outil de self-service pratique pour vous permettre de modifier votre Mot de passe SAP).
